Boss design isn't great
* holding attacks doesn't make boss fights more enjoyable
* very unpredictable mixups combined with this punishes those who read the boss -> maybe this was also an issue in Bloodborne, but wasn't noticeable due to the rally mechanic
* bullshit boss fights - a lot of people feel "I was lucky they didn't use this" which shouldn't be the case - FromSoft bosses have always been about feeling rewarded
* balance is off - for instance, magic is good against everything. Maybe this was an accessibility thing but it lacks one of my favorite aspects of Bloodborne where you really had to know what was good against each boss and use it. Instead, magic is good against everything. Maybe weapon arts need to be balanced too.
* I feel like boss design rewards using only a few OP weapons rather than fun builds.

Targeting system
* one thing common between all FromSoft games is hardest boss is the camera - much better in ER, but the targeting system needs a serious rework because it doesn't really work against some enemies. Good examples of this are massive mobs, larger enemies (such as dragons). Unlocking is a skill most players need and that's alright, but unlocked combat isn't great and could be limited with a better targeting system.
* I don't have a better solution to this, but I think it needs to be addressed. Honestly, the solution might be better unlocked combat.
* Horseback combate is absolutely terrible. I think this is because of the attack delays and moving on the back of the horse. I understand the attack delays when not mounted, but unfortunately it's super clunky when mounted. This isn't a game breaking experience, just something I noticed.

Multiplayer system needs a huge revamp
* Not talking about invasions/PvP
* The multiplayer system was designed in 2009 and definitely feels like it. Multiplayer (especially in an open world game like this) really needs to be reworked. Can't summon once you've beaten the boss? Trying to play with your friends only to have to resummon them constantly? There needs to be some sort of party system that's easier to use. It's 2022. 
* The previous invasion/co-op experience wasn't made for open world but the game is open world now so it needs to be fixed.
